1. (Done) Check stationarity for time series using Augmented Dickey-Fuller (ADF) Test
    (mean, variance and autocorrection must remain constant over time)

2. If the data is not stationary check for cointegration using Johanson's Cointegration Test
   "If two or more series are themselves non-stationary, but a linear combination
    of them is stationary, then the series are said to be cointegrated."
2.1 If the data is cointegrated use Vector Error Correction Model (VECM)
    VECM is suitable for non-stationary but cointegrated time-series
2.2 (Done) If the data is not cointegrated transform the time series to be stationary using differentiation. 
    Then proceed to use VAR model.
 
3. Select lag order(P) using information criteria: AIC and BIC 
   (Estimate a range of models and choose the best one)

4. Fit a VAR or VECM model with order(P)

5. Train the model of selected order(P)

6. For VAR model: invert transformations (such as differencing) previously performed on the data (if there were any)

7. Evaluate the model using test set
